# Film!

Film! — это сервис бронирования билетов в кинотеатр.

В нём можно:
- посмотреть список фильмов;
- прочитать краткую информацию о фильмах;
- узнать рейтинг фильма;
- увидеть доступные сеансы в кинотеатрах и оформить билеты.
